This folder contains my custom configuration for the **Windows 11 Taskbar Styler** (Windhawk mod).  
- **Mod Name:** Windows 11 Taskbar Styler  
- **Author:** m417z  
- **Process Hooked:** explorer.exe  
- **Version:** 1.5  

## About
The Windows 11 Taskbar Styler mod allows you to customize the Windows taskbar with community-contributed themes or your own creations.  

This setup uses the **DockLike** theme with advanced settings enabled, giving the taskbar a glassy blurred background, rounded corners, and subtle borders.

## Theme Used
- **Theme:** DockLike  
- **Custom Settings:** Advanced  
- **Config File:** [./Windhawk/config.txt](config.txt)  

## How to Use
1. Install [Windhawk](https://windhawk.net/).  
2. Install the **Windows 11 Taskbar Styler** mod.  
3. Copy the contents of [`config.txt`](config.txt) into Windhawk’s settings for the Taskbar Styler mod.  
4. Apply the theme in Windhawk to style your taskbar.  

---

💡 Tip: You can fork this repo and tweak [`config.txt`](config.txt) to experiment with your own styles.
